### Развёртывание ###
- Клонирование репозитория
```
git clone https://github.com/LASCode/toxin.git
```
- Установка зависимостей
```
npm install
```
- Запуск сервера разработки
```
npm serve
```
- Development сборка
```
npm dev
```
- Production сборка
```
npm prod
```
